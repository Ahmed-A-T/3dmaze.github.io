About the Project

The Maze Game is an engaging and challenging game developed using the C programming language and the SDL2 library. Navigate through intricate mazes, avoid obstacles, and reach the exit in the shortest time possible. This project showcases advanced programming techniques and efficient use of graphics libraries.

Features

Intuitive Controls

Easy-to-learn controls for seamless gameplay.

Challenging Levels

Multiple levels with increasing difficulty.

Engaging Graphics

Smooth and appealing visuals created with SDL2.

Cross-Platform

Compatible with multiple operating systems.
